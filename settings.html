OBS Integration

  • Configure your Kick username and Streamer.bot settings
  • Click "Save Configuration" to generate your bridge URL
  • Add a Browser Source in OBS with the generated URL
  • Bridge automatically syncs Kick chat and rewards to Streamer.bot

Streamer.bot Variables

Reward Variables

When a Kick reward is triggered, these variables are passed to your Streamer.bot action:

kickRewardTitle Reward name
kickUsername User who redeemed
kickUserInput User's text input
kickUserId User ID number
kickChannelId Channel ID number

Kick Variables

When viewers gift Kicks, these variables are passed to your Streamer.bot action:

kickGiftName Name of the gift
kickGiftAmount Amount of kicks gifted
kickUsername User who gifted
kickUserId Gifter's user ID
kickUserColor Gifter's username color
kickMessage Optional message with gift
