Event Types & Actions

Custom Rewards

Triggered when viewers redeem Kick rewards

Action Format
KickReward-Shoutout

Your Prefix + Reward Title

kickRewardTitle Reward name
kickUsername User who redeemed
kickUserInput User's text input
kickUserId User ID
kickChannelId Channel ID

Kicks Gifted

Triggered when viewers gift Kicks to your channel

Action Examples
Kicks1 Kicks10 Kicks50 Kicks100 Kicks500 Kicks1000 Kicks5000 Kicks10000

Action format: Kicks + Amount

kickGiftName Gift name
kickGiftAmount Amount gifted
kickUsername Gifter username
kickUserId Gifter ID
kickUserColor Username color
kickMessage Optional message

Hosts

Triggered when another streamer hosts your channel

Action Name
KickHost

Fixed action name

kickHostUsername Hosting streamer
kickHostViewerCount Viewer count
kickHostMessage Optional message
kickHostChatroomId Chatroom ID
